Update LibreWolf on Windows

Three ways to stay current: winget, Scoop, or manual installer. Verify your version and upgrade safely on Windows 11/10.

Check current version Update with winget Update with Scoop Manual update Rollback Next steps

Check your current version

In LibreWolf, go to Menu → Help → About LibreWolf. Compare with the latest version: 150.0.1‑1.

Update with winget (recommended)

Use the Windows Package Manager if you installed LibreWolf via winget or want to switch to managed updates.

  1. Open Windows Terminal as Administrator.
  2. Search for available package and version updates for LibreWolf.
  3. Apply the update when prompted.

Update with Scoop

If you installed via Scoop, updating is seamless and profile‑safe.

  1. Open a terminal (PowerShell).
  2. Check for updates in your buckets.
  3. Upgrade the LibreWolf package.

Manual update (installer)

Download the latest installer and run it over your existing installation. Your user profile is preserved.

Install guide
  • Close all running instances before updating.
  • Reopen About LibreWolf to ensure the version reads 150.0.1‑1.
  • If streaming breaks after update, see DRM & Streaming.

Rollback to a previous version

If a new release causes issues, consider rolling back until a fix ships.

  • Backup your profile folder first (about:profiles → Open Folder).
  • Uninstall LibreWolf (keep profile), then install the previous build.
  • Alternatively, use the portable build for testing.
