Step-by-step installation

  1. Run the downloaded installer. If prompted by SmartScreen, choose “Run anyway” for trusted builds.
  2. Choose the destination folder and confirm the install.
  3. Optionally set LibreWolf as your default browser via Windows Settings.
  4. Complete the setup and launch LibreWolf.
  5. Open “About LibreWolf” to confirm the version 150.0.1‑1.

Tip: Close other browsers during installation to import bookmarks or default-app preferences later without conflicts.

Verify the installer (recommended)

Compare the installer’s checksum with the published hash. Verifying ensures your download is intact and untampered.

  • Check the file name matches: librewolf-150.0.1-1-windows-x86_64-setup.exe
  • Compare the SHA256 hash against the official release notes.

First run: privacy checks

  • Telemetry is disabled by default; you don’t need to opt out.
  • If streaming sites don’t play, see DRM & Streaming to enable Widevine.
  • Want a minimal footprint? Consider the portable build.
