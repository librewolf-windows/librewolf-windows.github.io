Troubleshooting LibreWolf on Windows

Quick, practical fixes for common issues. Start with the basics below; if a problem persists, try a clean profile or portable build to isolate it.

LibreWolf won’t start / crashes on launch

  • Check for a stuck process in Task Manager; end tasks named LibreWolf.
  • Run in Safe Mode (hold Shift while launching) to disable add‑ons temporarily.
  • Create a new profile (about:profiles → Create) and test launch.
  • If it started after an update, roll back or test a portable copy.

Profile is locked or in use

  • Close all LibreWolf windows and end background processes in Task Manager.
  • Open your profile folder (about:profiles → Open Folder) and delete parent.lock if present.
  • Reboot Windows if lock files persist, then retry.

SmartScreen or antivirus warnings

  • Verify the installer filename: librewolf-150.0.1-1-windows-x86_64-setup.exe.
  • Optionally verify the SHA256 from the release notes.
  • If SmartScreen flags it, choose More info → Run anyway when you trust the source.

Network, SSL, or proxy errors

  • Check Windows date/time and certificate store updates.
  • Disable system‑wide proxies or VPNs temporarily to test.
  • Try about:networking#dns and clear DNS cache; restart the browser.
  • Test with a clean profile to exclude extension conflicts.

Streaming doesn’t play (DRM/Widevine)

DRM is off by default for privacy. Enable it only when needed.

  • Follow the DRM & Streaming guide to enable Widevine.
  • Disable strict blockers on the player domain if needed; refresh the page.

Slow performance or high CPU/RAM

  • Update to the latest build; fixes often land in new versions.
  • Reduce the number of extensions and heavy filter lists.
  • Check Task Manager → GPU process and drivers; update GPU drivers if outdated.
  • Test with Hardware Acceleration toggled to see if it helps on your system.

Fonts, UI scale, or DPI issues

  • Adjust Windows display scaling and restart the browser.
  • Check about:config overrides (e.g., layout.css.devPixelsPerPx) only if you know what you’re changing.
  • Install missing fonts or choose alternatives where sites require them.

Reset/refresh options

  • Backup your profile (about:profiles → Open Folder → copy to a safe place).
  • Create a fresh profile and test. Migrate bookmarks/passwords if stable.
  • As a last resort, uninstall and reinstall with the latest installer; keep your profile.

Import bookmarks, logins

  • Use the library/import tools to bring data from other browsers.
  • For password migration, export/import using CSV or a dedicated password manager.
